# YoungFolio

개인 포트폴리오 웹사이트 프로젝트입니다.  
디자인 감각과 기술 스택을 보여주기 위한 목적의 프로젝트예요.

---

## 🛠️ 구성

- main (projects) 
- about/ youngfolio
- about/ creator
- contact/ information
- contact/ inquiry (chat service)

## 🧰 기술 스택

| 구분 | 기술 |
|------|------|
| Frontend | React / Next.js / TypeScript |
| 스타일링 | CSS / Tailwind CSS / GSAP |
| 배포 | Vercel |
| 기타 | Supabase |

## 🚀 설치 및 실행 방법

```bash
# 저장소 클론
git clone https://github.com/JUYOUNG0728/youngfolio.git
cd youngfolio

# 의존성 설치
npm install
# 또는
yarn install

# 개발 서버 실행
npm run dev
# 또는
yarn dev
